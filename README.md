# GenAI Chatbot (Groq API)

This project is part of my Generative AI learning journey.

## What this project does
- Simple GenAI chatbot using Groq LLM
- Uses prompt engineering to control AI responses
- Demonstrates handling hallucinations and prompt drift
- Supports Tamil + English responses

## Why this project matters
This project demonstrates real-world GenAI skills such as:
- Working with free LLM APIs
- Prompt engineering and guardrails
- Debugging hallucinations
- Building production-ready AI workflows

## Tech Stack
- Python
- Groq API
- Prompt Engineering
- Virtual Environment (venv)

## Key Learnings
- API integration with LLMs
- Prompt guardrails and constraints
- Difference between GenAI and AGI
- Real-world debugging of model responses

## How to Run
1. Clone the repo
2. Create a virtual environment
3. Install dependencies:

pip install -r requirements.txt

4. Add your Groq API key in `.env`
5. Run:


python GenAI_Bot.py
